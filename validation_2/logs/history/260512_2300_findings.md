# Technical Finding: Etapa 1 — Estadísticos Robustos O3 × 3 niveles

**Context**: Validación de cálculos del Aplicativo PT, Etapa 1 (Estadísticos Robustos de Dispersión)

## The Discovery

Se implementaron y ejecutaron exitosamente los scripts de validación para estadísticos robustos (mediana, MAD, MADe, nIQR) sobre las 3 combinaciones primarias de O3.

**Hallazgos clave:**

1. **R y Python coinciden exactamente** (diff < 1e-15) para todas las métricas, incluyendo cuartiles. Esto confirma que `quantile(type=7)` de R y `numpy.percentile` de Python producen resultados idénticos con n=13.

2. **Los valores calculados coinciden con ptcalc**: Se verificó que `calculate_mad_e()` y `calculate_niqr()` del paquete ptcalc producen exactamente los mismos valores que los scripts independientes.

3. **La columna `sample_1` del dato pivoteado** (primera réplica) es correcta para los estadísticos robustos — confirmado leyendo app.R líneas 596–610.

4. **Factores confirmados contra ISO 13528:2022 §9.4**: MADe = 1.483 × MAD, nIQR = 0.7413 × IQR.

5. **Los helpers compartidos** (`helpers.R`, `helpers.py`) fueron creados en `validation_2/` con COMBOS limitados a O3 × 3 niveles.
