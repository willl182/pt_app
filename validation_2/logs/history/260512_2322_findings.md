# Technical Finding: Etapa 2 Homogeneidad — Validación PT

**Context**: Validación de cálculos de homogeneidad ISO 13528:2022 §9.2 para O3 × 3 niveles
**Timestamp**: 260512_2322
**Estado**: COMPLETADA ✅

## The Discovery

Se crearon desde cero los scripts de Etapa 2 (Homogeneidad) en `validation_2/`, reemplazando los scripts antiguos que referenciaban `validation/helpers.R` (inexistente).

### Arquitectura de los scripts

1. **18 métricas numéricas** por combo (g, m, general_mean_homog, x_pt, s_x_bar_sq, s_xt, sw, sw_sq, ss_sq, ss, sigma_pt, MADe, u_sigma_pt, nIQR, criterio_c_MADe, criterio_exp_MADe, criterio_c_nIQR, criterio_exp_nIQR)
2. **4 evaluaciones cualitativas** por combo (ss_vs_c_MADe, ss_vs_c_nIQR, ss_vs_exp_MADe, ss_vs_exp_nIQR)
3. **Verificación contra ptcalc**: Los valores R coinciden exactamente con `calculate_homogeneity_stats()` (diff = 0 para todos los estadísticos clave)
4. **R vs Python**: max diff = 1.14e-13, todas PASS con tolerancia 1e-9

### Discrepancia conocida: `calculate_homogeneity_criterion_expanded`

Existen **2 firmas** de esta función:

| Fuente | Args | Fórmula |
|--------|------|---------|
| ptcalc | 2 (σ_pt, u_σ_pt) | `0.3*σ×√(1+(uσ/σ)²)` |
| R/app.R | 3 (σ_pt, sw, g) | `F1×(0.3×σ)² + F2×sw²` con tabla F1/F2 |

Los scripts de validación usan la versión de 3 args (coincide con app.R, que es la referencia).

### Resultado de homogeneidad

ss > c para los 3 niveles O3 en todos los criterios (MADe, nIQR, simple and expandido). Esto es **esperado** para gases de calibración que presentan variabilidad entre muestras.

## Learnings

1. **Los scripts antiguos** (`validation/stage_02_*.py`) referencian helpers inválidos y usan `canonical_row` que no existe en `validation_2/helpers.py`. Los nuevos scripts evitan esta dependencia.
2. **El script Python usa numpy** para la matriz de datos, pero cálculos estadísticos manuales (var ddof=1, median) para evitar discrepancias de redondeo.
3. **La columna `sample_id`** en el DataFrame pivotado causa problemas con el filtro `sample_*` — se debe excluir explícitamente (`c.replace("sample_", "").isdigit()`).
