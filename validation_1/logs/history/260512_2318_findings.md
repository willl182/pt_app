# Technical Finding / Milestone

**Context**: Fase 6 - Algoritmo A detallado

## The Discovery

El Algoritmo A convergía en una sola iteracion para los combos O3 del subconjunto
`summary_n13.csv`, por lo que la traza utile necesitaba incluir explicitamente la
fila `initial` para no perder el estado de arranque del algoritmo.

La implementacion quedó alineada entre R y Python con las mismas columnas de traza:
`iteration`, `step`, `x_median`, `x_mad`, `sigma`, `x_w_median`, `x_w_mad`,
`sigma_w`, `max_abs_z`, `converged`, ademas de los valores original y winsorizado.
