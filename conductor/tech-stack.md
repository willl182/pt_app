# Technology Stack

## 1. Core Language
*   **R:** The primary programming language for all statistical logic and application development.

## 2. Web Framework
*   **Shiny:** Used for building the interactive web application.

## 3. Data Manipulation & Visualization
*   **Tidyverse:** A collection of R packages (including `dplyr`, `tidyr`, `ggplot2`) used for data science tasks.
*   **Plotly:** Used for creating interactive plots.

## 4. Reporting
*   **RMarkdown:** Used for dynamically generating reports (PDF/Word).

## 5. UI Components
*   **DT:** R interface to the DataTables library for interactive tables.
*   **rhandsontable:** Used for Excel-like editable tables within the application.
