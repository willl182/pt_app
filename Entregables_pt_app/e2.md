## 3. Subagente Designado
*   **Nombre del Subagente:** `api_auditor`
*   **Rol:** *Function Documenter & API Auditor*
*   **Responsabilidades:**
    1.  Escanear el código de R para extraer funciones declaradas, identificando patrones roxygen2.
    2.  Actualizar la base de datos de firmas de funciones (`funciones_extraidas.csv`) ante cualquier cambio en el código base.
    3.  Asegurar la trazabilidad metrológica asociando las funciones estadísticas a las cláusulas correspondientes de la norma **ISO 13528:2022**.

## 4. Procedimiento de Verificación y Pruebas
Para regenerar la lista de funciones y ejecutar el test de concordancia:

```r
# Ejecutar la extracción de firmas
source("Entregables_pt_app/02_funciones_usadas/R/lista_funciones.R")

# Ejecutar la prueba de consistencia
source("Entregables_pt_app/02_funciones_usadas/tests/test_02_firma_funciones.R")
```

El test valida que:
1.  Todas las funciones registradas en el CSV existan en los entornos cargados.
2.  Los argumentos de cada función coincidan exactamente con la firma documentada.
