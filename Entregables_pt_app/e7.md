## 1. Objetivo del Entregable
Incorporar el componente de visualización de datos al aplicativo Shiny. Esto incluye la creación de gráficos interactivos de barras, histogramas de distribución de frecuencia, diagramas de control para puntajes y mapas de calor (heatmaps) globales usando paquetes como `ggplot2` y `plotly`, integrándolos en pestañas dinámicas organizadas por contaminante y nivel de concentración.

## 3. Subagente Designado
*   **Nombre del Subagente:** `viz_engineer`
*   **Rol:** *Data Visualization & Dashboard Engineer*
*   **Responsabilidades:**
    1.  Desarrollar y optimizar las visualizaciones interactivas de `plotly` para que tengan un renderizado fluido, tooltips limpios y escalas legibles en español.
    2.  Implementar la composición de gráficos multipanel mediante `patchwork` o layouts de Shiny.
    3.  Asegurar que los gráficos sigan la guía visual y de estilos (estilo premium de shadcn/ui con colores HSL armonizados y contrastes adecuados para la lectura técnica).

## 4. Procedimiento de Verificación y Pruebas
Para correr el script de verificación automatizada del módulo gráfico:

```r
# Ejecutar los tests de generación de gráficos
source("Entregables_pt_app/07_dashboards/tests/test_07_graficos.R")
```

El test comprueba:
1.  Que el archivo `app_v07.R` contenga código que instancie objetos de tipo `ggplot` o `plotly` de forma válida.
2.  Que los placeholders de gráficos en el UI correspondan con outputs reales definidos en el Server.
3.  La legibilidad sintáctica del archivo Mermaid `diagrama_flujo.mmd`.
