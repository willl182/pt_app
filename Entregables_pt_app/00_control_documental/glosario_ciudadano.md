# Glosario ciudadano del aplicativo PT

**Versión:** 1.0  
**Fecha:** 2026-07-14

Estas definiciones explican cómo se usan los términos en la documentación. No
reemplazan las definiciones normativas oficiales.

| Término | Explicación en lenguaje ciudadano |
|---|---|
| Analito | Sustancia, propiedad o característica que se mide en una muestra. |
| Categoría de desempeño | Lectura cualitativa de un puntaje: satisfactoria, cuestionable o no satisfactoria, según el tipo de puntaje y sus límites. |
| Ensayo de aptitud (PT) | Comparación organizada de resultados entre participantes para evaluar su desempeño frente a un criterio definido. PT proviene de *proficiency testing*. |
| Estabilidad | Comprobación de que el ítem no cambió de manera relevante durante almacenamiento, transporte o periodo de análisis. |
| Homogeneidad | Comprobación de que los ítems distribuidos son suficientemente semejantes para la finalidad de la ronda. |
| Incertidumbre de medida | Información numérica que expresa la duda razonable asociada a un resultado. No significa que el resultado sea incorrecto. |
| Ítem de ensayo de aptitud | Material, muestra o artefacto preparado y distribuido para que el participante realice la medición. |
| Participante | Laboratorio u organización que reporta resultados en la ronda. |
| Puntaje En | Comparación entre el resultado participante y el valor asignado considerando incertidumbres expandidas. |
| Puntaje z | Diferencia entre el resultado participante y el valor asignado, expresada en unidades de la desviación estándar para evaluación. |
| Puntaje z' | Variante de z que también considera la incertidumbre del valor asignado. |
| Puntaje zeta | Diferencia frente al valor asignado considerando las incertidumbres estándar del participante y del valor asignado. |
| Ronda | Ciclo completo del ensayo de aptitud: preparación, distribución, recepción de resultados, evaluación e informe. |
| Sigma PT (σpt) | Desviación estándar elegida para evaluar el desempeño. Define la escala con la que se calcula, por ejemplo, el puntaje z. |
| Valor asignado | Valor usado como referencia para evaluar los resultados de los participantes. Puede proceder de una referencia, consenso u otro método documentado. |

## Regla editorial

La primera aparición de un término técnico en cada documento debe enlazar o
remitir a este glosario y, si afecta una decisión, incluir una explicación
breve en el mismo párrafo. Los símbolos (`σpt`, `u`, `U`) deben acompañarse de
su nombre y unidad cuando corresponda.
