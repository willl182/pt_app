# Matriz requisito–documento–evidencia

La fuente estructurada es `matriz_trazabilidad.csv`. Contiene una fila mínima
para cada entregable E01–E09 y permite responder:

1. ¿De dónde procede el requisito?
2. ¿Qué fuente Markdown lo desarrolla?
3. ¿Qué evidencia lo respalda y cuál es su estado?
4. ¿Qué limitación impide declararlo cerrado?

Los requisitos actuales son provisionales porque proceden del plan
`260712_1639`, no de un contrato/TDR/acta primaria disponible. Los registros
que apuntan a evidencia futura se conservan como `Pendiente`; una ruta futura
no debe interpretarse como archivo existente ni como prueba ejecutada.
