# Fase 4 completada — Cadena de Incertidumbre

**Context**: Validacion downstream de Algoritmo A — Etapa 4 cadena de incertidumbre

## Lo que se implemento

### Scripts implementados

#### R (`validation/stage_04_uncertainty_chain.R`)
- Algoritmo A de winsorización iterativa (ISO 13528:2022 Anexo C)
- Cálculo de nIQR
- Cálculo de cadena de incertidumbre para 4 métodos:
  1. Referencia (x_pt de homogeneidad, sigma_pt de homogeneidad)
  2. Consenso MADe (mediana, sigma_pt = 1.483 * MADe)
  3. Consenso nIQR (mediana, sigma_pt = nIQR)
  4. Algoritmo A (Algoritmo A winsorizado)
- Cálculo de 7 métricas por método: x_pt, sigma_pt, u_xpt, u_hom, u_stab, u_xpt_def, U_xpt
- Output: `stage_04_uncertainty_chain_r.csv` (intermedio)

#### Python (`validation/stage_04_uncertainty_chain.py`)
- Implementación idéntica a R
- Algoritmo A de winsorización iterativa
- Cálculo de nIQR
- Cálculo de cadena de incertidumbre para 4 métodos
- Comparación con resultados R
- Output: `stage_04_uncertainty_chain_py.csv` (intermedio)
- Output: `stage_04_uncertainty_chain.csv` (tabla canónica)
- Output: `stage_04_uncertainty_chain_report.md`

## Observaciones

- **u_stab = 0 para todos los combos**: diff_hom_stab = 0 en este dataset (datos stab = datos hom)
- **u_hom = ss de homogeneidad**: consistente con fórmula de app.R
- **u_xpt = 1.25 * sigma_pt / sqrt(n_part)**: fórmula de app.R
- **u_xpt_def = sqrt(u_xpt^2 + u_hom^2 + u_stab^2)**: propagación de incertidumbre combinada
- **U_xpt = k * u_xpt_def (k=2)**: incertidumbre expandida
- **Algoritmo A converge para todos los combos**: no hay errores de convergencia
- **Diferencias numéricas mínimas**: orden de 1e-15 a 1e-17, muy por debajo de tolerancia

## Fórmulas validadas

```
u_xpt = 1.25 * sigma_pt / sqrt(n_part)
u_hom = ss (de homogeneidad)
u_stab = Dmax / sqrt(3) (de estabilidad)
u_xpt_def = sqrt(u_xpt^2 + u_hom^2 + u_stab^2)
U_xpt = k * u_xpt_def (k=2)
```
