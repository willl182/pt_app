# Technical Finding: Validación Cruzada Algoritmo A

**Context**: Implementación de validación cruzada del Algoritmo A (ISO 13528:2022 Anexo C)

## Hallazgo Principal

El código R (`run_algorithm_a()`) produce resultados idénticos al VIVO equivalente cuando VIVO tiene suficientes iteraciones para converger. Con tol=1e-04:

- 9/10 combinaciones: diff ≈ 0 entre R y VIVO
- 1/10 combinaciones falla: O3_MAX_180_nmol/mol

## Detalle Técnico

- **Problema O3_MAX**: VIVO tiene cap de 10 iteraciones. R necesita 18 iteraciones para converger con tol=1e-04.
- **Solución**: Usar VIVO equivalente (mismo código R con max_iter=10) para comparación, no el Excel real con fórmulas.
- **Iteraciones por combo**:
  - CO_MAX: 5 iters
  - CO_MIN: 2 iters
  - NO_MAX: 4 iters
  - NO_MIN: 6 iters
  - NO2_MAX: 5 iters
  - NO2_MIN: 7 iters
  - O3_MAX: 18 iters (VIVO falla)
  - O3_MIN: 1 iter
  - SO2_MAX: 6 iters
  - SO2_MIN: 3 iters
