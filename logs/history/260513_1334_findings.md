# Technical Finding / Milestone

**Context**: Algoritmo A y nuevo metodo de sigma_pt por laboratorios expertos

## The Discovery

El flujo de puntajes de la app ya estaba estructurado por metodos a traves de `score_combo_info`, `global_combo_specs` y `compute_scores_for_selection()`. Eso permitio agregar `Expertos (4)` sin reestructurar la app: basto con introducir una funcion `calculate_expert_sigma_pt()` y conectarla en los puntos donde se calculan `x_pt`, `sigma_pt` y `u(x_pt)`.

Para el Algoritmo A, cuando `s*_0` calculado por MADe es cero, usar `stats::sd(values)` como valor inicial efectivo permite continuar con las iteraciones ISO en lugar de abortar el proceso si la muestra todavia tiene dispersion aritmetica distinta de cero.
