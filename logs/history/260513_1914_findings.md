# Technical Finding / Milestone

**Context**: Fase 7 del plan `excel-formulas-validacion-o3`.

## The Discovery

Los heat maps globales no requieren calculos estadisticos nuevos. Son una vista
derivada de `puntajes_EA`: metodo, tipo de score, participante, nivel, valor
redondeado a dos decimales y evaluacion textual.

La implementacion mas estable para los libros con formulas es usar referencias
directas por indice:

- `puntajes_EA` mantiene orden metodo x participante.
- `heatmap_datos_globales` usa orden metodo x score x participante.
- `heatmap_global` enlaza cada celda visible a la fila correspondiente de
  `heatmap_datos_globales`.

Esto evita formulas `MATCH` sobre columnas completas, reduce fragilidad y
mantiene auditable que el heatmap solo reorganiza datos existentes.
