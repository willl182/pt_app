# Technical Finding / Milestone

**Context**: Preprocesador CALAIRE solo referencia

## The Discovery

El pipeline CALAIRE quedó implementado para procesar únicamente datos de referencia. La entrada de ronda (`data/raw/datos_ronda.csv`) no tiene fila de unidades y usa columnas `CO_ref`, `SO2_ref`, `CO_gen`, `SO2_gen`; por eso `read_calaire_raw()` ahora detecta si existe fila de unidades y `clean_calaire_raw()` normaliza columnas `*_ref`/`*_gen`.

El criterio de hora válida se cambió de `n = 60` a `n >= 45` (75%) para ronda, estabilidad y homogeneidad. La incertidumbre horaria se calcula como `u = s / sqrt(n)`.

Para ronda se agregaron salidas de referencia:

- `data/processed/h_referencia_ronda.csv`
- `data/processed/referencia_ronda.csv`

La consolidación final usa promedios horarios: nivel 0 conserva una hora; niveles no-cero conservan hasta 3 horas. Esto recupera la tercera hora parcial de CO 2.8 / SO2 80 (`n = 57`) y excluye la cuarta hora de CO 1.4 / SO2 40.
