# Technical Finding / Milestone

**Context**: O3 validation report generation

## The Discovery

The final validation report works better as a generated artifact than as a hand-edited Markdown file. By consolidating the canonical stage reports into a single R script, the report stays aligned with the actual validation outputs and avoids drift.

The report now includes:

- formulas for Stage 4 value assignment
- formulas for Stage 5 scores
- a consolidated scores table by method
- consistent 4-decimal formatting

## Key Files Affected

- `validation_1/generate_informe_validacion_o3.R`
- `validation_1/validation/informe_validacion_o3.md`
- `validation_1/outputs/stage_04_uncertainty_chain_report.md`
- `validation_1/outputs/stage_05_scores_report.md`
