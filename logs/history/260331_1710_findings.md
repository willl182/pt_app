# Implementacion Fase 3 completada — Estabilidad

**Context**: Validacion downstream de Algoritmo A — Etapa 3 estabilidad

## Lo que se implemento

### R (`validation/stage_03_stability.R`)
- Carga de datos desde `stability_n13.csv` con `load_wide_data()`
- Lectura de resultados de homogeneidad (stage_02_homogeneity_r.csv)
- Carga de valores raw de homogeneidad para u_hom_mean
- Cálculo de 13 métricas de estabilidad:
  - g, m, general_mean_stab, x_pt_stab, s_x_bar_sq_stab, sw_stab
  - ss_sq_stab (abs()), ss_stab, diff_hom_stab
  - u_hom_mean, u_stab_mean, criterio_simple, criterio_expandido
- Output: `stage_03_stability_r.csv` (intermedio)

### Python (`validation/stage_03_stability.py`)
- Carga idéntica a R
- Lectura de resultados de homogeneidad (stage_02_homogeneity_py.csv)
- Carga de valores raw de homogeneidad para u_hom_mean
- Comparación tripartita leyendo CSV intermedio de R
- Generación de `stage_03_stability.csv` (tabla canónica)
- Generación de `stage_03_stability_report.md`

## Observaciones

- **diff_hom_stab = 0 para todos los combos**: los datos de estabilidad son idénticos a los de homogeneidad (mismo CSV fuente)
- **u_hom_mean y u_stab_mean**: calculados con sd de TODOS los valores (no solo medias)
- **criterio_expandido**: usa fórmula de estabilidad (c + 2*sqrt(u_hom^2 + u_stab^2)), NO la tabla F1/F2
- **ss_sq usa abs()**: consistente con Fase 2

## Flujo de ejecución

```bash
Rscript validation/stage_03_stability.R     # Paso 1: genera CSV intermedio R
python3 validation/stage_03_stability.py     # Paso 2: genera comparación + reporte
```
