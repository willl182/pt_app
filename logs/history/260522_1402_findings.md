# Technical Finding / Milestone

**Context**: Script organization for `pt_app`

## The Discovery

The `scripts/` directory can be split cleanly by execution role.

- `scripts/aplicativo/` contains the four scripts that `app.R` invokes
  directly in the Shiny workflow.
- `scripts/adicionales/` contains demos, batch preprocessors, generators,
  validation wrappers, and other supporting utilities that are not called
  directly from the UI.

The app code only needed path updates after the move; the scripts themselves
did not require functional changes for this reorganization.
