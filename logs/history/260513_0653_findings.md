# Technical Finding / Milestone

**Context**: PT validation report formatting

## The Discovery

The Stage 5 report is better presented as a compact reference block rather than a full participant matrix when the goal is a final validation document. A single participant per combo keeps the report readable while still preserving the numeric cross-check.

For the final validation report, manual evidence is the right place to represent Excel and aplicativo outputs when those values are not exported as separate fields in the Stage 5 CSV. Explicit placeholders make the missing evidence visible and auditable.
