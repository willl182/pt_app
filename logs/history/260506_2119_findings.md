# Technical Finding / Milestone

**Context**: Procesamiento de ensayo para `part_1`

## The Discovery

Se creó un flujo separado para procesar `data/raw/datos_ronda_part.csv`, equivalente al flujo de referencia de ronda pero aplicado al participante de ensayo `part_1`.

El archivo contiene datos minutales con columnas:

- `CO_p1`
- `SO2_p1`
- `CO_gen`
- `SO2_gen`

Se agregaron mapeos de columnas `*_p1` en `clean_calaire_raw()` y una función nueva `compute_hourly_averages_participant_ronda()` que usa la misma lógica de bloques consecutivos de 60 minutos por nivel generado, con hora válida si `n >= 45`.

## Outputs

- `data/processed/h_part_1_ronda.csv`: salida horaria.
- `data/processed/part_1_ronda.csv`: salida consolidada por contaminante/nivel.

Resultado de ejecución:

- 902 filas crudas.
- 901 filas tras limpieza.
- 26 horas válidas.
- 10 niveles consolidados.
