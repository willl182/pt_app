# Technical Finding / Milestone

**Context**: POC GPT53CDX - Implementacion de pipeline de validacion tripartita

## The Discovery

Pipeline completo de validacion downstream post-Algoritmo A implementado en `validation/val3/`:

1. **`poc_gpt53cdx_val.R`**: Script R con dos modos (`--mode app_r_excel`, `--mode merge_py`). Replica logica de `app.R` (capa APP) y reimplementa calculos independientemente (capa R). Genera 5 workbooks Excel + master CSV.

2. **`poc_gpt53cdx_val.py`**: Script Python sin dependencias externas (solo stdlib: csv, math, collections). Reimplementa todos los calculos: MADe, nIQR, Algoritmo A, homogeneidad, estabilidad, incertidumbres, puntajes z/z'/zeta/En.

3. **Flujo de 3 pasos**: app_r_excel -> python -> merge_py produce CSV maestro con 17 columnas canonico.

4. **Workbooks**: estructura INDICE + 15 hojas combo + RESUMEN por cada seccion (Robust Stats, Homogeneity, Stability, Uncertainties, Scores).

## Key Design Decisions

- `u_stab = d_max / sqrt(3)` incondicional (igual que app.R)
- `ALGO_A_TOL = 1e-04` (igual que app.R)
- Niveles 0: sigma_pt cercano a 0 produce NA en puntajes (esperado)
- Python sin numpy/pandas: usa funciones manuales para quantile, median, mean, sd
