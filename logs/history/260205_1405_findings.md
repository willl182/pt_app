# Technical Finding: Verificación Cálculos Auditoría CO 0-μmol/mol

**Context**: Auditoría de cálculos de homogeneidad y estabilidad ISO 13528:2022

## The Discovery

### 1. Cálculos Básicos Correctos
Los cálculos de promedio general, sx (SD de promedios de muestras), y sw (SD intra-muestras) del archivo de auditoría **coinciden exactamente** con los cálculos manuales usando las fórmulas ISO 13528:2022.

### 2. Problema con σpt
El valor σpt = 0.00579 reportado en la auditoría **NO es calculable** desde los datos usando métodos estándar ISO 13528:

| Método | Valor Calculado |
|--------|-----------------|
| MADe (todos valores) | 0.04009 |
| MADe (promedios) | 0.00162 |
| nIQR (todos valores) | 0.04014 |
| nIQR (promedios) | 0.00142 |
| **Auditoría reporta** | **0.00579** |

**Hipótesis:** El σpt = 0.00579 es un valor prescrito externamente, probablemente:
- Incertidumbre del patrón de referencia (cilindro de gas)
- Valor definido por el organizador del ensayo de aptitud
- Error de transcripción de otro análisis

### 3. Implicación para Criterio de Homogeneidad
Con ss = 0.01786 y σpt = 0.00579:
- Criterio: ss ≤ 0.3 × σpt = 0.00174
- Resultado: 0.01786 >> 0.00174 → **NO PASA**

Si el σpt fuera calculado correctamente (ej. MADe = 0.04009):
- Criterio: ss ≤ 0.3 × 0.04009 = 0.01203
- Resultado: 0.01786 > 0.01203 → **Tampoco pasaría**

### 4. Problema en Implementación del App
En `R/pt_homogeneity.R:89-96` la implementación actual NO sigue ISO 13528:

```r
# Código actual (INCORRECTO):
abs_diff_from_xpt <- abs(sample_data[, 2] - x_pt)  # Solo columna 2
sigma_pt <- stats::median(abs_diff_from_xpt)
MADe <- 1.483 * sigma_pt

# Debería ser (según ISO 13528):
all_values <- as.vector(sample_data)
MADe <- 1.483 * median(abs(all_values - median(all_values)))
```
