# Technical Finding / Milestone

**Context**: PT validation report evidence without Excel

## The Discovery

When the Excel workbook is not usable, the validation report can remain
auditable by rendering the canonical `app_value` evidence generated from the
same calculation path as `app.R`. The report now lists R, Python, and
`Aplicativo app.R` rows directly in each O3 evidence table and marks Excel as
not used rather than leaving screenshot placeholders.

The evidence formatter must group repeated metrics by `section`; otherwise
Stage 4 and Stage 5 collapse to the first method only. For documentation
stability, the formatter falls back to `r_value` when `app_value` or
`python_value` is empty/NaN.
