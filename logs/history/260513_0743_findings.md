# Technical Finding / Milestone

**Context**: PT validation report evidence layout

## The Discovery

When the validation report needs to carry horizontal screenshots, the most
usable layout is a row-oriented evidence table. Keeping `Scripts R/Python`,
`Excel`, and `Aplicativo` as separate rows inside each O3 block preserves the
comparison context without forcing the screenshots into narrow columns.

This keeps Stage 4 aligned with the user's workflow: compare what the scripts
produce, then attach the matching Excel and aplicativo captures in the same
section.
