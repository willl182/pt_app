# Technical Finding / Milestone

**Context**: Validation stage 01 in `validation_1`

## The Discovery

La primera etapa de validacion puede aislarse con solo `data/summary_n13.csv`, filtrando `participant_id != "ref"` y tomando `mean_value` por contaminante/nivel. Para `O3_0`, `O3_80` y `O3_180`, las metricas robustas quedan consistentes entre R y Python cuando:

- `MADe = 1.483 * MAD`
- `nIQR = 0.7413 * IQR`
- los cuartiles usan `type = 7`
