# Technical Finding / Milestone

**Context**: Integración de referencia CALAIRE en `app.R`

## The Discovery

La referencia CALAIRE procesada quedó integrada en la interfaz y el análisis principal de la app sin mover cálculo puro a Shiny ni procesar datos de participantes dentro del preprocesador.

La integración se hace en `app.R`:

- `calaire_reference_df()` lee `data/processed/referencia_ronda.csv`.
- `use_calaire_reference` permite activar/desactivar el uso de CALAIRE como referencia.
- `pt_prep_data()` reemplaza filas `participant_id == "ref"` por filas CALAIRE solo cuando existe coincidencia exacta de `pollutant` y `level`.
- `compute_scores_for_selection()` usa `u_value` de CALAIRE como `u_xpt` del método de referencia.
- La pestaña "Valor de referencia" ahora muestra la referencia usada, la tabla consolidada CALAIRE y el detalle horario.

## Important Constraint

No se implementó remapeo automático de niveles. Si `summary_n13.csv` usa niveles como `0-μmol/mol` y `referencia_ronda.csv` usa `0-ppm`, solo se reemplazan combinaciones con coincidencia exacta. Esto evita asignar valores de referencia a niveles incorrectos.
