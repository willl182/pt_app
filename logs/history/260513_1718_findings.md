# Technical Finding / Milestone

**Context**: Excel formulas validation for O3

## The Discovery

The validation pipeline now has a regenerated snapshot and a workbook layer
with base data sheets. The snapshot generator must remain the source of truth
for `resultado_homogeneidad`, `resultado_estabilidad`, `valor_asignado`,
`algoritmo_A`, `puntajes_EA`, and `informe_global`.

The current workbook generator only covers:

- raw data sheets for homogeneity, stability, participants, and reference
- the frozen `validacion_snapshot`
- a high-level `validacion_final` control sheet

The scientific formulas are still missing and should be implemented in the next
phase.
